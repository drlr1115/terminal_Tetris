# terminal_Tetris
python based Tetris game in linux terminal

Usage: Tetris
Options:
-h, --help            show this help message
Operating Instructions:
KEY             ACTION
w/i     --      ROTATE
s/k     --      DOWN
a/j     --      LEFT
d/l     --      RIGHT
SPACE   --      DROP
p       --      PAUSE/RESUME
+       --      LEVEL UP
-       --      LEVEL DOWN
q/ESC   --      QUIT

Config.py
Configurations

Utils.py
Definition of Tetris shape, key mapping, util funtion/class
